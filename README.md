# Statistical Assessment of Potentially Habitable Exoplanets

A notebook-based data analysis project that combines exoplanet catalog data, uncertainty-aware classification, and a simplified habitability screening pipeline. The workflow starts with cleaning NASA exoplanet archive data, then estimates flux, classifies planets by composition using Bayesian MCMC, studies correlations among key physical variables, and finally filters rocky planets using gravity, flux, and estimated surface temperature.

## Project Summary

This project asks a blunt but useful question: which exoplanets might be remotely compatible with Earth-like surface conditions? To answer it, the notebook builds a full analysis chain:

* clean and deduplicate the catalog
* compute incident flux for planets with enough stellar/orbital information
* classify planets into rocky, ice giant, and gas giant groups using Bayesian inference
* study correlation structure with Pearson and Spearman metrics
* estimate surface temperature with a greenhouse-style model
* filter for planets that fall inside a conservative habitability window

The final output is not a confirmation of life, because reality refuses to be that neat. It is a ranked, uncertainty-aware shortlist of planets that deserve further attention.

## What the Notebook Does

### 1. Data cleaning and preparation

The notebook loads the exoplanet catalog, removes duplicate planet entries, converts numeric fields, and keeps only physically meaningful rows. For repeated entries, it retains the record with the lowest uncertainty or the most recent update, depending on the section of the analysis.

### 2. Flux estimation

For planets with the required orbital and stellar parameters, the notebook computes an estimated incident flux using an idealized blackbody-style model. This flux is then used as a proxy for the stellar energy received by the planet.

### 3. Error handling and uncertainty propagation

Measurement errors in flux, mass, radius, and gravity are propagated explicitly. The analysis keeps track of relative uncertainty so that unstable or poorly constrained planets do not dominate the conclusions.

### 4. Bayesian MCMC planet classification

A power-law mass-radius model is used to separate planets into broad classes. The notebook uses MCMC sampling to infer posterior probabilities for the classes rather than forcing hard labels from a single threshold.

The classes used are:

* Rocky
* Ice Giant
* Gas Giant

A second classification stage adds density into the inference, which improves separation and makes the result less naive.

### 5. Correlation analysis

Pearson and Spearman correlation heatmaps are used to study relationships among mass, radius, density, flux, gravity, temperature, and class labels. The notebook highlights that mass and radius are not equally informative, and density is often the stronger separator between planet types.

### 6. Surface temperature estimation

For rocky planets, the notebook estimates surface temperature using a greenhouse-inspired model with:

* albedo
* atmospheric opacity
* surface pressure
* gravity
* incoming flux

Because atmospheric data are not generally available, these quantities are treated probabilistically and inferred with MCMC using Bayesian approach.

### 7. Habitability screening

The final stage filters planets using a conservative Earth-like window:

* surface temperature between 263 K and 333 K
* gravity between 0.5 g and 2.0 g
* flux between 0.5 and 5.0 Earth units

In the current catalog sample used in the notebook, no planet satisfies all of these conditions simultaneously.

## Interpretation Notes

This project is a statistical screening exercise, not a definitive astronomy verdict. The temperature model is simplified, atmospheric parameters are not directly observed for most planets, and the habitability thresholds are intentionally conservative. The output should be treated as a shortlist for follow-up study, not as proof of habitability.

## Limitations

* Uses simplified physical models
* Treats some atmospheric parameters as latent variables
* Depends heavily on catalog completeness and measurement quality
* Habitability conclusions are only meaningful under Earth-like assumptions
* Results are sensitive to the chosen thresholds for temperature, flux, and gravity
